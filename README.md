# lab1-udv
Домашнее задание №1: “Экран анкеты”

Сверстать страницу анкеты с полями:
 1. Фамилия
 2. Имя
 3. Отчество
 4. Возраст (Slider + Stepper, синхронизированы)
 5. Номер телефона (текст красный, при длине 11 символов становится зелёным)
 6. Пол (RadioButton, выбор только одного значения)
 7. Согласие с условиями (Switch)
 8. Кнопка «Отправить» (активна только при включённом Switch).

У каждого поля должен быть заголовок.
